Lightning Strategies is Chief Bitcoin Officer (CBO) as a Service — helping self-funded middle-market employers beat healthcare inflation with Bitcoin treasury and Strike BLOC strategies.

## Contents

- `index.html` — homepage (live)
- `index-v2.html` — next iteration in progress
- `manifesto.html` — "Fix the Money, Fix the World" philosophy page
- `the-bitcoin-blueprint.html` — free DIY Bitcoin onboarding guide
- `LS Logos/` — brand system (icons, lockups, wordmarks)
- `Bolt icon /` — lightning bolt icon exports
- `push.sh` — helper script to stage, commit (conventional format), and push changes

## Pushing changes

From inside this folder:

```bash
# Interactive — prompts for type and message
./push.sh

# One-liner
./push.sh feat "add advisory council section"
./push.sh fix  "correct hero CTA link"
./push.sh docs "update README"
```

Conventional commit types: `feat`, `fix`, `docs`, `style`, `refactor`, `chore`.
